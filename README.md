# brainfuck
Brainfuck interpreter in C, without using pointers

## Compiling:

```
gcc brainfuck.c -o brainfuck
```

## Example: HelloWorld!

```bash
./brainfuck ">>>--------<,[<[>++++++++++<-]>>[<------>>-<+],]++>>++<--[<++[+>]>+<<+++<]<<[>>+[[>>+<<-]<<]>>>>[[<<+>.>-]>>]<.<<<+<<-]>>[<.>--]>.>>."
```

## Example: Bubble sort

```
./brainfuck ">>,[>>,]<<[[<<]>>>>[<<[>+<<+>-]>>[>+<<<<[->]>[<]>>-]<<<[[-]>>[>+<-]>>[<<<+>>>-]]>>[[<+>-]>>]<]<<[>>+<<-]<<]>>>>[.>>]" asddsadfsafasfd

```
